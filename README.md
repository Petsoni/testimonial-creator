# EasyTestimonial

Web app for creating professional testimonial cards instantly.

## Features

- **Instant Generation**: Create polished testimonial cards in seconds
- **No Design Skills Required**: Pre-built templates and layouts
- **Export Ready**: Download as images
- **Customizable**: Adjust colors, layouts to match brand

## Use Cases

- Landing pages
- Portfolio websites
- Marketing materials
- Social media posts
- Sales presentations
- Case studies

## Tech Stack

- Next.js / React
- TailwindCSS
- Canvas API for image export
- Vercel deployment

## Getting Started

```bash
# Clone and install
git clone https://github.com/yourusername/easytestimonial.git
cd easytestimonial
npm install

# Development
npm run dev

# Production build
npm run build
```

## Usage

1. Navigate to `/builder`
2. Enter testimonial text
3. Add customer name and details
4. Customize design
5. Export as image 

## Deployment

Deployed on Vercel at [easytestimonial.xyz](https://easytestimonial.xyz)

```bash
# Deploy
vercel --prod
```

## Environment Variables

```env
NEXT_PUBLIC_APP_URL=https://easytestimonial.xyz
```
